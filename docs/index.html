Turn compiler noise into clean JSON

ZeroTest runs any command and converts its errors into structured diagnostics — file paths, line numbers, severity, error codes. Ready for your terminal or your AI agent.

⬇ Download View on GitHub →

What it does

🧹 Parse anything

mypy, tsc, gcc, rustc, javac, eslint, go build, Ruby — regex-based parsers for 8 languages and growing.

📋 Structured output

Every error becomes a JSON object with file, line, column, severity, error code, and message.

🖥 CLI mode

Run from any terminal: zerotest python -m mypy src/app.py. JSON goes to stdout.

🤖 MCP server

Works as an MCP tool for Claude, Cline, Continue, and any MCP-compatible agent.

🔍 Auto-detection

Detects language and toolchain from the command name and output. Override with hints if needed.

⚡ Single binary

Written in Go. Zero runtime dependencies. Drop-in binary for Mac, Linux, or Windows.

How to use

💻 CLI mode
🤖 MCP server

Run any command and get JSON diagnostics straight to your terminal.

# Python / mypy
zerotest python -m mypy src/app.py

# Go build
zerotest go build ./...

# TypeScript
zerotest tsc --noEmit

# C
zerotest gcc main.c -o main

# Rust
zerotest cargo build

# Flags
zerotest -h     # help
zerotest -v     # version

JSON is printed to stdout. Exit code matches the wrapped command.

Run as an MCP server for your AI agent. Just launch with no arguments:

zerotest

Then add to your MCP client config:

# Claude Desktop (claude_desktop_config.json)
{
  "mcpServers": {
    "zerotest": {
      "command": "/path/to/zerotest",
      "args": []
    }
  }
}
